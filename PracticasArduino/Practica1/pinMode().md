# pinMode()


## Descripción
**Configura el pin especificado para que se comporte como entrada o salida**

## Sintaxis
```c
pinMode(pin, modo);
```

## Parámetros
**Pin:** el número de pin Arduino para establecer el modo.

**Modo:** **INPUT**, **OUTPUT**
