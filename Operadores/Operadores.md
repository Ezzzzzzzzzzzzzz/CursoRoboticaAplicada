# Operadores

Los **operadores** son funciones a las cuales se les ha asignado un símbolo que indican cómo se manipulan los datos. 

Se pueden clasificar en los siguientes grupos: 

>- aritméticos,
>- relacionales 
>- lógicos
>- unitarios,
>- a nivel de bits
>- de asignación, 
>- operador condicional
>- etc.
